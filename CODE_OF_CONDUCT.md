# Code of Conduct

## Scope

This project is a security audit tool. All contributors and users must operate within legal and ethical boundaries.

## Rules

1. **Authorized use only** — H4RD must only be used on devices you own or have explicit written authorization to test
2. **Responsible disclosure** — If you discover a vulnerability in a hardware wallet, report it to the manufacturer before public disclosure
3. **No malicious use** — Do not use H4RD or its exploit modules to steal funds, compromise devices you don't own, or cause harm
4. **Respect privacy** — Do not share scan results, extracted keys, or blockchain data from devices belonging to others
5. **Constructive collaboration** — Be respectful in issues, PRs, and discussions

## Enforcement

Violations of this code of conduct may result in:
- Removal of contributions
- Ban from the repository
- Reporting to relevant authorities if illegal activity is suspected

## Contact

Report concerns via GitHub private messaging to the repository maintainer.
